RedactIPv4Addresses(ref string)

ATTEMPTS to replace IPv4 address(es) with RedactionReplacement. Should only be called if ShouldRedactLogs is enabled. Case-by-case redaction should be preferred for performance reasons over using this function. This function is intended for third-party messages (e.g., BattlEye) that we don't have control over.
