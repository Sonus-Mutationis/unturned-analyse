ConsumeKeyDown(KeyCode)

Should be used anywhere that Input.GetKeyDown opens a UI.

Each frame one input event can be consumed. This is a hack to prevent multiple UI-related key presses from interfering during the same frame. Only the first input event proceeds, while the others are ignored.
