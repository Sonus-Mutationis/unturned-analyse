GetHwids()

20-byte SHA1 salted hashes of client's hardware ID(s). Providing multiple HWIDs makes it more difficult to bypass HWID bans because spoofing a single component only changes one of the bans. For example spoofing the MAC address will not spoof the Windows GUID.

Randomized if system did not support hwid, or perhaps player is cheating. Should not be called on the client side, but just in case there is a default zeroed array.
